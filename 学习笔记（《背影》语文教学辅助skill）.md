# 学习笔记（《背影》语文教学辅助skill）

# 一、技能描述

本技能名为「学习笔记」，核心功能是辅助语文课文《背影》的教学工作，适配初中语文教学场景（兼顾教师教学引导、学生自主学习及课后巩固）。技能围绕《背影》的文本解析、情感主旨、重点词句、写作手法、课文结构等核心教学要点展开，以互动问答、重点讲解、思路引导的方式，帮助教师高效开展课堂教学，辅助学生理解课文内涵、掌握重点知识、体会父爱的深沉，同时梳理课文重难点，补充相关背景知识，解决教学及学习中的常见疑问，助力提升《背影》的教学质量和学习效果。

适用人群：初中语文教师、初中学生；使用场景：课堂教学互动、课后知识点巩固、课文预习、重难点答疑、作文素材积累。

# 二、触发词

触发词：/newslearn（在对话界面输入该触发词并发送，即可启动技能，进入《背影》教学辅助互动模式）

# 三、使用方法

1. 启动技能：在对话窗口输入触发词「/newslearn」，发送后技能自动启动，系统将发送欢迎语及核心功能引导，明确教学辅助的主要方向。

2. 选择使用场景/学习内容：启动后，系统将列出3类核心选项，用户（教师/学生）可回复对应序号选择：① 课文解析（段落大意、结构梳理）；② 重难点突破（重点词句、写作手法）；③ 情感主旨与拓展（父爱内涵、素材积累）；④ 答疑解惑（针对课文疑问提问）。

3. 互动教学/学习：根据选择的内容，系统将进行详细讲解，讲解过程中设置互动提问，引导用户思考；用户可随时打断，回复具体疑问，系统将针对性解答；教师可借助互动环节引导课堂讨论，学生可通过问答巩固知识点。

4. 切换/结束技能：学习/教学过程中，回复「切换内容」，系统将重新列出选项；回复「结束学习」，技能退出，系统将发送知识点总结，方便后续回顾。
